Understanding the Python Traceback
==================================

https://realpython.com/python-traceback/

Python wypisuje informację zwrotną , gdy w kodzie zostanie zgłoszony wyjątek. Wynik śledzenia może być nieco przytłaczający, jeśli widzisz go po raz pierwszy lub nie wiesz, co ci mówi. Jednak śledzenie Pythona zawiera mnóstwo informacji, które mogą pomóc w zdiagnozowaniu i naprawieniu przyczyny zgłaszania wyjątku w kodzie. Zrozumienie, jakie informacje dostarcza śledzenie w Pythonie, jest niezbędne, aby stać się lepszym programistą Pythona.
